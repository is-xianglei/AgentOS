🌊 Streamable HTTP 流式输出 Demo

传输协议: application/x-ndjson  |  客户端: fetch + ReadableStream  |  方向: POST 请求体 → 换行分隔 JSON 流

与 SSE Demo 的区别：
SSE: GET /stream?prompt=… | 协议: text/event-stream | 客户端: EventSource（浏览器原生，自动重连）
HTTP: POST /stream + JSON body | 协议: application/x-ndjson | 客户端: fetch 手动读取字节流
点击按钮开始，会看到 thinking / text / tool_use 三种数据块逐步出现…
